HacktoberFest

Where 4 PRs == Goodies

1. Register at hacktoberfest.digitalocean.com

2. Head over to github.com/thisisshub/HacktoberFest

3. Click on Add file then Create new File

4. Make a file with name enrollmentnumber_yourname.md, write something in it and then click propose a new file

5. Add 4 more such lines one-by-one with individual commits
